# vue-yodify

> A Vue.js component module

# Usage

    <vue-yodify text="Hello World!"></vue-yodify>

## Installation

### Using yarn

`yarn add vue-yodify`

### Using npm

`npm i --save vue-yodify`

## Demo and Docs

`npm run serve`
